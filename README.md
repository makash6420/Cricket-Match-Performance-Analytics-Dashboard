# Cricket-Match-Performance-Analytics-Dashboard

## Project Overview

This project analyzes cricket match data to evaluate player performance, team statistics, venue trends, and toss impact on match outcomes. The solution combines Python, SQL, and Power BI to deliver end-to-end sports analytics and interactive business intelligence dashboards.

## Objectives

* Analyze player batting and bowling performance
* Evaluate team-wise match statistics
* Study toss impact on match results
* Identify venue-specific performance trends
* Create interactive dashboards for data-driven insights

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* MySQL
* Power BI
* Microsoft Excel

## Dataset

The dataset contains:

* Match Information
* Team Details
* Player Statistics
* Toss Results
* Venue Information
* Match Outcomes

## Project Workflow

1. Data Collection and Cleaning
2. Data Transformation using Python
3. SQL Query-Based Analysis
4. Exploratory Data Analysis (EDA)
5. KPI Creation and Dashboard Development
6. Insight Generation

## Key KPIs

* Total Matches
* Win Percentage
* Total Runs Scored
* Total Wickets Taken
* Toss Win Impact
* Venue Performance Metrics

## Dashboard Features

* Team Performance Analysis
* Player Ranking Dashboard
* Toss Impact Analysis
* Venue Analysis
* Match Summary Dashboard

## Key Insights

* Identified top-performing players based on runs and wickets.
* Analyzed the relationship between toss outcomes and match results.
* Compared team performances across different venues.
* Highlighted patterns influencing match outcomes.
